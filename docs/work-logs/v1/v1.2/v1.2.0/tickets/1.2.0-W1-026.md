## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->

只改 hook + 測試，未動其他檔案。

1. `SKIP_PATTERNS` 補三個 sink 檔名 regex：`app_spacing\.dart$` / `app_typography\.dart$` / `terminal_constants\.dart$`。
   原因：017 skip list 僅含 ui_* 命名，但 013/014 ANA 選定的集中化 sink 是 app_*/terminal_*，
   致 018/020 將寫入的常數定義檔本體被 presence hook 誤攔，阻塞 bootstrap 自身。
2. `_STRING_EXCLUDE_CONTEXT` 補 `\bArgumentError\b` 與 `\btoString\s*\(` 兩個脈絡。
   原因：017 交付時觀察到參數校驗錯誤訊息與 toString 除錯輸出屬開發者面字串，被當 user-facing 過度偵測。
3. 補 3 個測試：`test_skip_sink_definition_files`、`test_string_in_argumenterror_excluded`、`test_string_in_tostring_excluded`。
