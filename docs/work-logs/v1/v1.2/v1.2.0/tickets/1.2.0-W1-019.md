## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
### 實作內容

建立 Flutter gen-l10n i18n 設施並遷移 013 清單的 user-facing 文字。

**設施建置**
- app/l10n.yaml：gen-l10n 設定（arb-dir=l10n, output AppLocalizations 至 lib/l10n）
- app/l10n/app_en.arb：英文 SOT，22 個三段式 camelCase key（含 description）
- app/pubspec.yaml：新增 flutter_localizations + intl 依賴、generate: true
- app/.gitignore：忽略 gen-l10n 產出 lib/l10n/app_localizations*.dart
- app/lib/main.dart：MaterialApp.router 接線 localizationsDelegates + supportedLocales + onGenerateTitle

**文字遷移（20/21 直接遷移 + 1 deferred）**
- main.dart appTitle（onGenerateTitle）
- home_screen.dart：homeAppBarTitle / homeHeadline / homeConnectButton
- qr_scanner_screen.dart：enrollmentScanTitle / commonDismiss
- enrollment_screen.dart：enrollmentOverwriteTitle / enrollmentOverwriteMessage / commonCancel / enrollmentReplaceButton / enrollmentSaveSuccess / enrollmentTitle / enrollmentInstruction / enrollmentScanButton
- terminal_screen.dart：terminalConnecting / terminalDisconnected / terminalReconnect(x2) / terminalErrorGeneric / terminalErrorAuth / terminalErrorTimeout / terminalErrorNetwork（_errorDisplayMessage 改收 AppLocalizations 參數）

**deferred 項目（建 follow-up 1.2.0-W1-027）**
- local_auth_biometric_service.dart authBiometricReason：位於不持有 BuildContext 的服務層，呼叫鏈 ConnectionManager.connect 亦無 context，跨域遷移超出 string-only 範圍。ARB key 已預留，原始碼加註解標記，建 1.2.0-W1-027 追蹤（quality-baseline 規則 5）。

### 範圍紀律
僅改文字字串，未動數字/顏色（020/021 範圍）。string-only 變更與 number/colour-only 變更可乾淨並存。

### 測試調整
- 新增 test/features/l10n/app_localizations_test.dart：驗證 supportedLocales 含 en + 19 個關鍵 key 載入正確英文值
- enrollment_screen_test.dart / terminal_screen_test.dart 的 buildTestApp() 補 localizationsDelegates + supportedLocales；find.text 英文斷言維持有效（en 為預設 locale）
