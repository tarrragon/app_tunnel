Round 1 三 reviewer 零重疊 frame，**全 A 級、零必修**：

| Reviewer | Frame | 評級 | finding |
|----------|-------|------|---------|
| A basil-writing-critic | 寫作規範 + 字句層 | A | 0 嚴重；建議 A1（其實連用）、A2（場景→情境） |
| B general-purpose | fact-check（web 佐證） | 通過 | 0 事實錯誤；6 宣稱 5 正確 + 1 細微（SDT 時序）；建議 B1/B3/B4 |
| C general-purpose | Zettelkasten + 讀者旅程 | A | 0 嚴重；9 跨卡連結逐一驗證正確、互連三角完整、原子性達標 |

修法（全建議採納，blog commit 1a11402）：
- B1：false-positive 補 SDT 正式理論 1950s 成形
- B3：type-i-error α 此消彼長補「樣本數固定」前提
- B4：type-i-error Type I/II 編號順序標明「純助記非歷史定義」
- A1：false-positive 「其實」連用 → 「並不存在」
- A2：抽象「場景」→「情境」
