## Problem Analysis
<!-- Schema[IMP/Problem Analysis]: 選填（小型 IMP 可留 placeholder；大型 IMP 建議填寫）（.claude/pm-rules/ticket-body-schema.md） -->
### 問題根因

TerminalScreen 的 _buildTerminalView()（connected 狀態）只有 status bar + terminal renderer + input field + toolbar，沒有 disconnect 或 back 操作按鈕。用戶進入 connected 狀態後無法主動斷線或返回首頁。

### 影響範圍

- app/lib/features/terminal/screens/terminal_screen.dart：_buildTerminalView() 和 _buildStatusBar() 需修改
- app/test/features/terminal/screens/terminal_screen_test.dart：需新增測試

### 現狀盤點

idle/connecting 已有 back 按鈕（共用 _buildConnectingView）。
connected 狀態無 back、無 disconnect。
disconnected/error 已有 back + reconnect。

### Context Bundle

ConnectionManager.disconnect() 已實作（line 102-105）。
實作方向：_buildStatusBar() 右側加 disconnect IconButton，呼叫 widget.connectionManager.disconnect()。
disconnect 後由 state stream 自動切到 disconnected view（已有 back）。
i18n：純 icon 免 i18n，或 AppLocalizations 加 disconnect 文字。
