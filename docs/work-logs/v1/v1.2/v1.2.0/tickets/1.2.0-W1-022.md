來源 ANA 1.2.0-W1-014 方案 C（深色「儀表艙」）IMP-2 落地。

### 1. 全域深色 ThemeData

- 新建 `app/lib/core/theme/app_theme.dart`：`AppTheme.dark()` 工廠，套 AppColors/AppTypography token 組 ColorScheme（cobalt primary + amber secondary + 中性墨色）、scaffold/canvas 背景、AppBar、filledButton、textTheme。顏色一律引用 AppColors 常數，零行內硬編碼。
- `main.dart`：`theme: ThemeData(colorSchemeSeed: Colors.blue, useMaterial3: true)` → `theme: AppTheme.dark()`。

### 2. 統一主操作按鈕

- 新建 `app/lib/shared/widgets/primary_action_button.dart`：`PrimaryActionButton`（底層 FilledButton，可選 icon），視覺由 theme 的 filledButtonTheme 集中承載。
- 3 處主操作按鈕統一為此 component：
  - home `FilledButton.icon` → PrimaryActionButton（保留 `Key('connect_terminal_button')`）
  - enrollment `ElevatedButton.icon` → PrimaryActionButton
  - terminal `ElevatedButton.icon` x2（disconnected + error view）→ PrimaryActionButton（保留 `Key('reconnect_button')`）
