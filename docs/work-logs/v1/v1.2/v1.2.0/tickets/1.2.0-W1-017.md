## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
新增 .claude/hooks/dart-presence-detection-hook.py（PreToolUse on Edit/Write/MultiEdit of .dart）。

偵測三類缺席（針對 greenfield 從不 bootstrap i18n/theme 的 presence-blind 盲區，015 根因）：
1. 硬編碼 user-facing 字串（CJK 或含空白多字英文；排除 log/debugPrint/print/assert/import/註解/annotation/throw Exception/Key()）
2. 裸 Color(0x...) / Colors.xxx（排除 UIColors/AppColors/Theme.of/colorScheme）
3. 魔術數字（SizedBox/EdgeInsets/fontSize/BorderRadius.circular/Duration；排除 UISpacing/UIFontSizes/UIBorderRadius/UIDurations）

與既有 hook 區隔（非重複）：
- style-guardian：PostToolUse + WARNING-only + 假設設施存在 + 掃整檔。本 hook：PreToolUse + blocking-with-override + 偵測缺席 + 只掃變更內容。
- l10n-sync：只在 .arb 編輯觸發（presence-blind）。本 hook：在 .dart 寫入即攔截硬編碼。

防呆（避免癱瘓 021/024/025 remediation）：
- 只掃變更內容：Edit 取 new_string、Write 取 content、MultiEdit join 各 new_string；本次未觸及的既有舊行不被重複攔截。
- override marker：命中行自身或前一行含 // presence-exempt / i18n-exempt / color-exempt / magic-exempt / style-exempt 即豁免。
- 降級路徑：PRESENCE_HOOK_MODE=warn 退化為純警告（exit 0 + allow），供 warn→觀察→升 block 漸進部署；預設 block（exit 2 + permissionDecision deny）。
- 範圍跳過：.g.dart/.freezed.dart/test/l10n/generated/theme 設施本體。

註冊：settings.json PreToolUse Edit 與 Write matcher 各追加一筆（JSON 已驗證）。
