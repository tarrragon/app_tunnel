# Execution Log

## Task Summary

移除 X-App-Tunnel-Token 認證閘道，簡化為透明反向代理；clientIP 改為只用 RemoteAddr；更新稽核 log 事件；重寫測試

## Problem Analysis
<!-- Schema[IMP/Problem Analysis]: 選填（小型 IMP 可留 placeholder；大型 IMP 建議填寫）（.claude/pm-rules/ticket-body-schema.md） -->
### Context Bundle

**遷移背景**：Cloudflare Tunnel → Tailscale mesh VPN。proxy 不再做認證。

**改動檔案**：`server/main.go`, `server/main_test.go`

**main.go 改動清單**：
1. 移除 `tokenHeader` 常數和 `X-App-Tunnel-Token` 相關程式碼
2. 簡化 `authProxyHandler`：移除 token 比對，改為直接透明轉發（可改名 `proxyHandler`）。移除 `crypto/subtle` import、`r.Header.Del(tokenHeader)`
3. 簡化 `clientIP`：只用 `r.RemoteAddr`，移除 `Cf-Connecting-Ip` 和 `X-Forwarded-For`
4. 移除 `runServe` 中 `secret.Load` 及相關 flag（secret-backend/secret-file/keychain-service/keychain-account/secret-env）。移除 `internal/secret` import
5. 保留 audit log：所有連線記 `connection accepted`。ErrorHandler 的 `connection failed` 保留
6. 保留 timeout、graceful shutdown 不變

**main_test.go 改動清單**：
1. 刪除舊測試：TestAuthGate_NoToken_404、TestAuthGate_WrongToken_404、TestAuthGate_GoodToken_Forwarded_AndStripsToken
2. 新增：TestProxy_TransparentForward、TestProxy_PreservesHeaders（Authorization 等透明轉發）、TestProxy_AuditLog、TestProxy_UpstreamDown_502
3. 更新 TestClientIP：只測 RemoteAddr（移除 CF header 測試）

**殘留檢查**：無 `X-App-Tunnel-Token`、`Cf-Connecting-Ip`、`crypto/subtle`、`internal/secret` 引用

## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->

- `authProxyHandler` 重命名為 `proxyHandler`，移除 token 驗證邏輯，改為直接透明轉發
- `clientIP` 函式移除，直接使用 `r.RemoteAddr`
- 移除 `crypto/subtle`、`internal/secret` 依賴及所有 secret 相關 CLI flags
- 刪除 3 個 auth gate 測試 + 2 個 clientIP 測試；新增 `TestProxy_TransparentForward`、`TestProxy_PreservesAuthorizationHeader`、`TestProxy_BackendDown_502`、`TestIsWebSocketUpgrade`
