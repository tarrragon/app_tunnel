## Problem Analysis
<!-- Schema[IMP/Problem Analysis]: 選填（小型 IMP 可留 placeholder；大型 IMP 建議填寫）（.claude/pm-rules/ticket-body-schema.md） -->
### Context Bundle

**目標**：建立手機終端機特殊按鍵工具列，改善 CLI 打字體驗。

**規格參考**：SPEC-004 FR-04（input 端）

**實作要求**：
1. TerminalKeyboard widget：固定在終端機畫面底部的工具列
2. 按鍵列表：Esc / Tab / Ctrl（toggle 模式）/ 方向鍵（上下左右）
3. Ctrl 組合鍵：Ctrl 按下後 toggle 高亮，下一個按鍵自動加 Ctrl 修飾（Ctrl+C = 0x03、Ctrl+D = 0x04 等）
4. 所有按鍵事件轉為 ttyd input 格式：callback 回傳 Uint8List（'0' prefix 由 caller 處理）
5. Widget 測試：按鍵 tap → 正確 callback 值

**按鍵對應**：
- Esc → 0x1B
- Tab → 0x09
- 方向鍵：Up=\x1B[A, Down=\x1B[B, Right=\x1B[C, Left=\x1B[D
- Ctrl+C → 0x03, Ctrl+D → 0x04, Ctrl+Z → 0x1A

**目錄**：app/lib/features/terminal/keyboard/
