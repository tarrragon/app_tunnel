## Problem Analysis
<!-- Schema[IMP/Problem Analysis]: 選填（小型 IMP 可留 placeholder；大型 IMP 建議填寫）（.claude/pm-rules/ticket-body-schema.md） -->
### Context Bundle

**目標**：建立終端機文字渲染器，將 WS 收到的 ttyd output 渲染成終端機畫面。

**實作要求**：
1. TerminalRenderer widget：接收 Stream<String> 終端機輸出，渲染文字
2. ANSI escape sequence 基本支援：SGR 色彩（前景/背景 8+16 色）、游標移動（CUU/CUD/CUF/CUB）、清除（ED/EL）
3. 終端機緩衝區：固定行數緩衝（如 1000 行）+ 自動捲動到底部
4. 使用 monospace 字型、黑色背景、綠色/白色前景
5. Widget 測試：驗證文字渲染 + ANSI 色彩解析

**建議**：可用 CustomPainter 或 RichText + TextSpan 實作。初版不需完整 xterm 模擬，先支援基本文字 + 色彩即可。

**目錄**：app/lib/features/terminal/renderer/
